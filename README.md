# CodeGym AI: Your AI-Powered Coding Practice Arena 🚀

CodeGym AI is a modern, interactive web application built with Next.js that helps you practice and master your coding skills. It leverages cutting-edge AI technology to provide a dynamic, personalized learning experience with real-time feedback, progress tracking, and an intelligent coding assistant.

## ✨ **New Features & Improvements**

### 🎯 **Enhanced Learning Experience**
- **Monaco Code Editor**: Professional-grade code editor with syntax highlighting, line numbers, and language-specific features
- **Smart Problem Generation**: AI-powered coding challenges that adapt to your skill level and learning path
- **Progressive Hints System**: Get help when you're stuck without spoiling the solution
- **Learning Paths**: Structured progression through different programming concepts and difficulty levels

### 🤖 **AI-Powered Features**
- **Intelligent Code Analysis**: Get detailed feedback on code quality, best practices, and optimization suggestions
- **Smart Code Execution**: AI-based code evaluation with detailed output analysis
- **Conversational AI Assistant**: Ask questions, get hints, and receive personalized guidance
- **Adaptive Problem Difficulty**: Problems that adjust based on your performance and learning pace

### 📊 **Progress Tracking & Gamification**
- **Comprehensive Progress Dashboard**: Track your learning journey across languages and difficulty levels
- **Achievement System**: Unlock badges and milestones as you progress
- **Streak Tracking**: Maintain daily coding habits with streak counters
- **Performance Analytics**: Detailed insights into your solving speed and success rates

### 🎨 **Modern UI/UX**
- **Responsive Design**: Works seamlessly across all devices and screen sizes
- **Beautiful Animations**: Smooth transitions and micro-interactions
- **Dark/Light Mode Support**: Choose your preferred theme

## 🚀 **Getting Started**

### Prerequisites
- [Node.js](https://nodejs.org/) (version 20 or later)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- Google Gemini API key

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone <your-repository-url>
   cd CodeGym-AI
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env.local` file in the root directory:
   ```bash
   # Required: Google Gemini API Key
   GOOGLE_AI_API_KEY=your_actual_api_key_here
   ```
   
   **Get your API key from:** [Google AI Studio](https://aistudio.google.com/app/apikey)

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to [http://localhost:9002](http://localhost:9002)

## 🛠 **Tech Stack**

- **Framework**: [Next.js 15](https://nextjs.org/) with App Router
- **AI Integration**: [Google Gemini via Genkit](https://firebase.google.com/docs/genkit)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/) with Radix UI primitives
- **Code Editor**: [Monaco Editor](https://microsoft.github.io/monaco-editor/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with custom design system
- **State Management**: React hooks and context
- **Type Safety**: [TypeScript](https://www.typescriptlang.org/)
- **Icons**: [Lucide React](https://lucide.dev/)
